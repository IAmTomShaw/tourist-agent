# Tourist Agent

A simple Streamlit frontend for a Travel Research Agent that helps users discover and explore travel options.

## Features

- **Run Agent Button**: Start the travel research process with a single click
- **Real-time Status Updates**: View the agent's progress as it researches travel options
- **Results Grid**: Browse available travel packages in an easy-to-read grid format
- **Interactive Selection**: Select from various travel options

## Installation

1. Clone the repository:
```bash
git clone https://github.com/IAmTomShaw/tourist-agent.git
cd tourist-agent
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running the Application

To start the Streamlit app, run:

```bash
streamlit run app.py
```

The app will open in your default web browser at `http://localhost:8501`.

## Usage

1. Click the "Run Agent" button to start the travel research process
2. Watch as the agent provides status updates on its progress
3. Once complete, browse through the available travel options displayed in a grid
4. Click "Select" on any option to choose it
5. Use "Start New Search" to begin a new research session
